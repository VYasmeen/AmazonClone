# AmazonClone

AmazonClone is a full-stack web application that replicates key features of an e-commerce platform similar to Amazon.  
It includes user authentication, product catalog browsing, shopping cart functionality, and order management, implemented using modern web technologies.

---

## Project Overview

This project demonstrates the development of a scalable e-commerce application with core functionalities such as:

- User registration and login  
- Product listing and search  
- Shopping cart operations  
- Checkout workflow  
- Order placement and management

The aim is to simulate an Amazon-like user experience while following clean architecture and best coding practices.

---

## Features

- User Authentication (Sign up, Sign in, Sign out)  
- Browse and Search Products  
- Add to Cart / Remove from Cart  
- Persistent Shopping Cart  
- Order Placement  
- Responsive UI
