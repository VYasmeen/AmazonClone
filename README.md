# AmazonClone 🛒

AmazonClone is a full-stack web application that replicates core features of an Amazon-like e-commerce platform. It includes authentication, product browsing, shopping cart functionality, and order workflows built using modern web technologies.

## Overview
This project demonstrates the development of a scalable e-commerce application with essential features such as user authentication, product discovery, cart management, and checkout flow. The goal is to simulate a real-world online shopping experience while following clean architecture and best practices.

## Features
- User Authentication (Sign up, Sign in, Sign out)  
- Browse and Search Products  
- Add to Cart / Remove from Cart  
- Persistent Shopping Cart  
- Order Placement Flow  
- Responsive UI
